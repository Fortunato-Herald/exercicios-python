# 📘 Exercícios em Python

Este repositório contém uma coleção de exercícios desenvolvidos em **Python**, com o objetivo de praticar lógica de programação, estruturas de controle, funções, manipulação de dados e resolução de problemas.

## 🎯 Objetivo

A proposta desta pasta é servir como material de estudo e prática para reforçar os conhecimentos em Python, desde conceitos básicos até exercícios um pouco mais elaborados.

## 📂 Estrutura

Os arquivos estão organizados em uma pasta contendo diferentes exercícios, que podem abordar temas como:

- Entrada e saída de dados
- Operações matemáticas
- Estruturas condicionais
- Laços de repetição
- Funções
- Listas, tuplas e dicionários
- Manipulação de strings
- Resolução de problemas práticos

## 🐍 Tecnologias utilizadas

- **Python 3**

## ▶️ Como executar os exercícios

1. Certifique-se de ter o **Python 3** instalado em sua máquina.
2. Abra o terminal na pasta do projeto.
3. Execute o arquivo desejado com o comando:

```bash
python nome_do_arquivo.py
